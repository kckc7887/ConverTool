# ConverTool v0.1.4

## 变更摘要

- 图标更新：使用新的 ConverTool 图标，已应用到程序窗口与可执行文件图标。
- 插件体验：插件管理支持拖拽安装（仅接受 `.zip`），主界面“添加插件”也限制为 `.zip`。
- 失败提示：插件安装失败时改为弹窗提示，并按当前语言显示具体错误原因。
- 插件管理增强：支持在 `manifest.json` 标注 `author`，在插件管理中显示（默认 `UnKown`）。
- 配置激活逻辑：主界面在“无输入文件”时不自动激活插件配置，显示“输入文件以激活插件配置”提示。
- 插件目标文案优化：
  - FFmpeg 与 ImageMagick 的目标格式名称改为更易懂的说明文本。
  - ImageMagick 目标格式移除重复的 `TIFF` 选项（仅保留一个）。

## 构建与发布

- 版本号：`0.1.4`
- 运行时：`net8.0`，`win-x64`
- 发布形式：
  - `full`（self-contained）
  - `lite`（framework-dependent）
