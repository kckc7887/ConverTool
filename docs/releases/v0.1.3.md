# ConverTool v0.1.3

## 变更摘要

- **可执行文件命名**：发布产物可执行文件改为 `ConverTool.exe`（任务管理器/窗口标题更贴近产品名）。
- **发布方式**：`net8.0` 目标平台 `win-x64` 同时提供两种包：
  - `full`：`self-contained`（随包携带 .NET 运行时）
  - `lite`：framework-dependent（不携带 .NET 运行时；需要用户机器安装 .NET 8 运行时）

## 下载选择：full vs lite

本版本在 GitHub Release 下同时提供两个 zip：

- 完整包（full）：`ConverTool-v0.1.3-win-x64.zip`
  - `self-contained`：用户无需安装 .NET 8 运行时
  - 包含符号/调试相关文件（例如 `*.pdb`），方便排查问题
- 轻量包（lite）：`ConverTool-v0.1.3-win-x64-lite.zip`
  - `framework-dependent`：不携带 .NET 运行时，体积更小（本地生成大小约 `10.70MB`）
  - 需要用户安装 .NET 8 运行时（建议安装 .NET 8 Desktop Runtime）
  - 会移除符号/调试相关文件（例如 `*.pdb` / `createdump.exe`），不影响正常使用

选择建议：
- 你只是想正常使用转换且目标机未安装 .NET 运行时：优先下载 `full`
- 你目标机已安装 .NET 8 运行时：优先下载 `lite`
- 你在意崩溃/兼容性排查、需要符号栈信息：下载 `full`
