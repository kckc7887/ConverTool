# ConverTool v0.1.3

## 变更摘要

- **可执行文件命名**：发布产物可执行文件改为 `ConverTool.exe`（任务管理器/窗口标题更贴近产品名）。
- **发布方式**：`net8.0` 目标平台 `win-x64` 改为 `self-contained`，用户无需手动安装 .NET 8 运行时即可运行。

## 下载选择：full vs lite

本版本在 GitHub Release 下同时提供两个 zip：

- 完整包（full）：`ConverTool-v0.1.3-win-x64.zip`
  - 包含符号/调试相关文件（例如 `*.pdb`），方便你排查问题
- 轻量包（lite）：`ConverTool-v0.1.3-win-x64-lite.zip`
  - 默认推荐：体积更小（本地生成大小约 `41.78MB`）
  - 会移除符号/调试相关文件（例如 `*.pdb` / `createdump.exe`），不影响正常使用

选择建议：
- 你只是想正常使用转换：优先下载 `lite`
- 你在意崩溃/兼容性排查、需要符号栈信息：下载 `full`

## 构建信息

- **运行时**：`net8.0`，`win-x64`，**self-contained**（随包交付 .NET 运行时）
- **产物目录**：`artifacts/host/v0.1.3/win-x64/`
- **压缩包**：
  - `artifacts/ConverTool-v0.1.3-win-x64.zip`（full）
  - `artifacts/ConverTool-v0.1.3-win-x64-lite.zip`（lite）
