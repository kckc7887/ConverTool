## 面向用户的变更

### 1. 新功能

#### 1.1 文档转换器插件（PandocDocumentTranscoder）
- **新增内置插件**：支持 Markdown、Word、HTML、EPUB、PDF 等多种文档格式互转
- 使用 Pandoc + LibreOffice（自动下载或系统 PATH）
- 支持pdf转图片（长图/多图压缩包）
- 支持输入：.md、.docx、.html、.epub、.pdf 等
- 支持输出：.docx、.pdf、.html、.epub、.md、.doc 等

### 3. 自定义命名模板标签
- 新增三个自定义标签
- 用户可在界面上自定义标签显示文本
- 自定义标签文本会持久化保存到用户设置
- 输出文件名中会使用自定义标签文本

#### 3.1 目标格式选择器 UI 优化
- 从传统 ComboBox 改为 **Button + Flyout + ListBox** 组合
- 下拉列表同时显示**格式名称**和**格式 ID**

#### 3.2 改进的用户体验
- 为不支持的文件格式添加明确的错误提示
- 优化命名模板状态计算，提升响应速度
- 语言切换时正确刷新所有命名模板相关文案

---

### 4. 修复的问题

| 问题 | 说明 |
|------|------|
| 命名模板在切换插件时重置 | 修复命名模板在切换插件时被覆盖的问题 |
| 命名模板重启后不保存 | 修复命名模板在应用重启后不恢复的问题 |
| 空模板自动选择 | 修复空模板时未能自动选中原文件名标签的问题 |
| 自定义标签不生效 | 修复自定义标签在输出文件名中不显示的问题 |
| 不支持格式无提示 | 修复输入不认识的文件格式时无适当提示的问题 |
| 工具下载失败 | 优化 ImageMagick、FFmpeg 等工具的下载源与提取逻辑 |

## 技术性变更

### 5. 代码结构重构

#### 5.1 ViewModel 职责分离
| 新增/重构的文件 | 说明 |
|-----------------|------|
| `Host/Services/ServiceLocator.cs` | 新增：服务定位器（DI 容器包装），管理 `I18nService`、`PluginCatalog`、`PluginI18nService` 单例 |
| `Host/Settings/SettingManager.cs` | 新增：统一配置管理，封装 Host 与插件的 `config.json` 读写 |
| `Host/ViewModels/InputFileViewModel.cs` | 新增：输入文件管理子 ViewModel，负责拖拽、浏览、不支持格式检测 |
| `Host/ViewModels/NamingTemplateViewModel.cs` | 新增：命名模板管理子 ViewModel，负责标签管理、自定义标签文本、状态计算 |
| `Host/ViewModels/ConfigFields.cs` | 重构：配置字段 ViewModel 集合独立文件 |

#### 5.2 `MainWindowViewModel.cs` 大幅重构
- 从 900+ 行拆分，通过组合子 ViewModel 降低复杂度
- 输入文件逻辑委托给 `InputFileViewModel`
- 命名模板逻辑委托给 `NamingTemplateViewModel`
- 配置字段逻辑封装进 `ConfigFields`

### 7. 持久化系统改进

#### 7.1 `UserSettingsFile` 新增字段
```csharp
public class UserSettingsFile
{
    // 原字段
    public string? Locale { get; set; }
    public string? OutputDir { get; set; }
    public bool UseInputDirAsOutput { get; set; }
    public string? NamingTemplate { get; set; }

    // ↓ 新增
    public string? CustomToken1Text { get; set; }
    public string? CustomToken2Text { get; set; }
    public string? CustomToken3Text { get; set; }
    public bool? EnableParallelProcessing { get; set; }
    public int? Parallelism { get; set; }
    public bool? KeepTemp { get; set; }
}
```

#### 7.2 插件配置 `persistValue`
- 在 `settings[]` 每项上支持可选字段 `persistValue: false`
- `false` 时仅从文件读取恢复界面，保存时不把 UI 覆盖写回 `value`
- 插件配置读写路径以 `PluginEntry.PluginDir/config.json` 为准

---

### 8. 全局服务与初始化

#### 8.1 `ServiceLocator` 替代直接静态字段
```csharp
// 旧：
AppServices.Plugins = PluginCatalog.LoadFromOutput(...)

// 新：
ServiceLocator.Initialize(AppContext.BaseDirectory)  // 创建单例
AppServices.ReloadPlugins(baseDir)                    // 内部调用 ServiceLocator
```

#### 8.2 `AppServices` 门面
- `I18n`、`Plugins`、`PluginI18n` 均通过 `ServiceLocator.GetService<T>()` 获取
- `AppServices.ReloadPlugins(baseDir)` 代理到 `ServiceLocator.ReloadPluginCatalog()`

---

                            <TextBlock Text="{Binding Text}" />

                    </DataTemplate>
                </ListBox.ItemTemplate>
            </ListBox>

    </Button.Flyout>

#### 9.2 事件处理
- `OnTargetFormatFlyoutListKeyDown`：回车/空格选中
- `OnTargetFormatFlyoutListPointerReleased`：点击选中并关闭 Flyout
