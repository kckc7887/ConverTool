# ConverTool Host v0.1.2

本版主要聚焦于：主机 UI 体验修正、配置字段能力增强、以及插件侧外部进程日志的可观测性提升。

## 变更摘要

- **UI 双窗布局**：左侧设置区滚动、右侧处理区保持固定，并在左右之间加入细分割线；隐藏视觉滚动条但保留鼠标/触控板滚动体验。
- **焦点体验**：点击非可聚焦区域时清理键盘焦点，避免 `TextBox`/`ComboBox` 残留选中态。
- **输入持久化行为**：用户在输入框中的输入路径不再持久化到用户设置中。
- **新增宿主字段类型 `Number`**：提供 “Spinbox 风格”控件（输入框 + 右侧上/下按钮）用于数值类配置（例如把插件 `crf` 从 Range 调整为 Number）。
- **插件管理器展示增强**：支持插件在 `manifest.json` 中配置 `titleKey` / `descriptionKey`，在插件列表中显示本地化后的标题与说明。
- **外部进程日志实时回传**：插件启动外部进程（如 `ffmpeg`）时，将 `stderr` 按行通过 `reporter.OnLog(...)` 实时写入主机处理日志（便于判断宿主/插件问题源头）。
- **插件安装逻辑复用**：新增 Host 侧 `PluginZipInstaller` 共享 zip 安装逻辑；新增 `UserSettingsStore` 负责用户设置 JSON 读写。
- **发布版本信息**：`Host` 与其依赖的 `ConverTool.PluginAbstractions` 均提升至 `v0.1.2`。

- **程序集信息版本**（`InformationalVersion`）：`0.1.2`
