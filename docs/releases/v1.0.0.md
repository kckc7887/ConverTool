# ConverTool v1.0.0

首个 **1.0** 正式版本：应用与契约包 `ConverTool.PluginAbstractions` 同步到 **1.0.0**，默认附带两个基础插件，并修复 ImageMagick 插件中文语言包无效导致目标格式显示英文的问题。

## 亮点

- **版本**：应用、发行用安装包与 `PluginAbstractions` 契约包均为 **1.0.0**（契约包另见 NuGet / Release 中的 `.nupkg`）。
- **内置插件**：`setup.exe`、zip 便携包及默认 `plugins/` 中均附带 **FFmpeg 视频转码**、**ImageMagick 图片转换**（详见根目录 `README.md` / `docs/README.md`）。
- **修复**：`imagemagick.image.transcoder` 的 `zh-CN.json` 为合法 UTF-8 JSON；插件 i18n 在解析失败时不再错误缓存空结果；切换界面语言时清空插件语言缓存。

## 发行物（Windows x64）

**普通用户不需要从源码构建任何东西。** 在 [GitHub Releases](https://github.com/kckc7887/ConverTool/releases) 下载下面任意一种即可安装或使用：

- 推荐：**`ConverTool-v1.0.0-setup.exe`** — 图形化安装向导（可选 Full / Lite、可选是否安装内置插件）。
- 或解压即用：**`…-full.zip`** / **`…-lite.zip`**（便携目录，内含 `ConverTool.exe`）。

以下为本次 Release **Assets** 中的文件说明：

- `ConverTool-v1.0.0-win-x64-full.zip`：自带 .NET 8 运行时，解压即用。
- `ConverTool-v1.0.0-win-x64-lite.zip`：依赖本机已安装的 .NET 8 Desktop Runtime。
- `ConverTool-v1.0.0-setup.exe`：Inno Setup 安装包（同上，**已预编译并随 Release 提供**）。
- `ConverTool.PluginAbstractions.1.0.0.nupkg`：插件契约包（可本地上传私有源，或自行 `dotnet nuget push` 到 nuget.org）。

## 升级说明

- 若你从源码或 CI 构建 Host，请先发布或本地打包 **`ConverTool.PluginAbstractions` 1.0.0**，再还原/编译 Host（`Host.csproj` 已引用该版本）。
