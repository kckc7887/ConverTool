# ConverTool v1.1.1 发布说明

**契约包** `ConverTool.PluginAbstractions` 仍为 **1.1.0**，未变更。

---

## 文档插件（`pandoc.document.transcoder`）逻辑修复

**PDF→PNG / JPG / ZIP** 会首次从 **nuget.org** 拉取 PDF 渲染相关依赖并解压到本地缓存。此前在 Zip 内匹配条目时使用了带前导 `/` 的路径片段，与包内真实条目名不一致，导致**几乎无法解压出所需文件**，缓存不完整、转换失败。

本版改为按包内实际路径（如 `lib/net8.0/`、`runtimes/win-x64/native/`）匹配，首次下载解压可正常完成。仍需能访问 **api.nuget.org**，或使用环境变量 **`CONVERTOOL_PDF_RENDER_DIR`** 指向已备好的完整目录。
