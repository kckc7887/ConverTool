# ConverTool v1.0.1

小版本更新：改进主界面**输入区**交互与文案，Host 版本 **1.0.1**。插件契约包 `ConverTool.PluginAbstractions` 现为 **1.0.1**（新增 `ConfigSchema.FieldPersistOverrides` / `FieldPersistOverrideRule` 等，用于声明「持久化快照覆盖」；与源码中的 `manifest.json` 字段 `fieldPersistOverrides` 对齐）。

## 亮点

- **输入**：由多行路径文本框改为 **选择/拖拽** 区域；已选文件以 **图标 + 文件名** 列表展示。
- **操作**：单项 **移除**、**清空全部**；处理完成后仍会**清空输入列表**（与 v1.0.0 行为一致）。
- **文案**：占位提示为「选择文件或拖拽到此」等；发行说明与文档表述同步修正。

## 发行物（Windows x64）

**普通用户**从 [GitHub Releases](https://github.com/kckc7887/ConverTool/releases) 下载即可，无需自行编译。

- 推荐：**`ConverTool-v1.0.1-setup.exe`**
- 或便携包：**`ConverTool-v1.0.1-win-x64-full.zip`** / **`ConverTool-v1.0.1-win-x64-lite.zip`**

## 升级说明

- 自 v1.0.0 升级：直接覆盖安装或解压新版即可；用户设置与插件目录逻辑不变。
